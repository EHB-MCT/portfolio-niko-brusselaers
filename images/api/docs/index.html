• GET:

    • /getRoomData

      returns object with result object room

    • /getAllRoomData

      returns object with result object rooms(arr(room))

  • POST:

    • /receiveRoomData

      post sensor data to database

    • /addRoom

      add a new room to the database

  • PUT:

    • /updateRoom

      update already existing room

  • DELETE:

    • /deleteRoom

      delete certain room and all it's data from database
